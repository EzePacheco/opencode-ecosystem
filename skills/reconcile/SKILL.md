# Reconcile Findings

Use this skill after review when the change needs correction or consolidation.

Workflow:

1. Start from explicit reviewer findings.
2. Apply the smallest correct fixes.
3. Resolve cross-builder inconsistencies and integration gaps.
4. Re-run relevant checks when possible.
5. Report applied findings, rejected findings, and residual risk.

Do not redesign the solution unless a validated finding proves the current
approach is invalid.
